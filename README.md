# 🍕 Pizza Sales Analysis Dashboard | Power BI

## 📌 Project Overview

This project is an interactive Power BI dashboard designed to analyze pizza sales data and uncover valuable business insights.

The dashboard provides a comprehensive view of revenue performance, customer ordering behavior, product popularity, and sales trends through interactive visualizations and KPIs.

---

## 🎯 Business Objectives

* Analyze overall revenue and sales performance.
* Identify peak ordering hours.
* Determine top-selling pizza categories.
* Analyze pizza size performance.
* Identify pizzas generating the highest revenue.
* Track customer ordering trends.
* Support business decisions through data-driven insights.

---

## 🛠 Tools Used

* Microsoft Power BI
* Power Query Editor
* DAX Measures
* CSV Dataset

---

## 📂 Dataset Information

The project uses four datasets:

* Orders.csv
* Order Details.csv
* Pizzas.csv
* Pizza Types.csv

These tables were connected through relationships to create a complete sales analysis model.

---

## 🧹 Data Cleaning Process

Data preparation was performed using Power Query Editor:

* Promoted Headers
* Changed Data Types
* Trimmed Unwanted Spaces
* Handled Missing Values
* Removed Duplicate Records
* Validated Data Consistency

---

## 📊 Dashboard Features

### KPI Cards

* Total Revenue
* Total Orders
* Total Quantity Sold
* Average Order Value

### Interactive Filters

* Month
* Pizza Category
* Pizza Size

### Visualizations

#### Revenue Trend Analysis

Tracks revenue trends over time.

#### Revenue by Pizza Category

Compares revenue generated by different pizza categories.

#### Pizza Size Performance Analysis

Identifies which pizza sizes contribute most to sales.

#### Peak Ordering Hours

Analyzes customer ordering behavior throughout the day.

#### Top Revenue Generating Pizzas

Highlights pizzas generating the highest revenue.

#### Detailed Product Analysis

Provides deeper insights into individual pizza performance.

---

## 🔍 Key Insights

* Peak order activity occurs during lunch hours.
* Certain pizza categories contribute significantly more revenue than others.
* Pizza size influences sales performance.
* Revenue trends vary across different periods.
* Interactive filtering enables detailed business analysis.

## 🚀 How to Use

1. Download the Power BI (.pbix) file.

3. Refresh the dataset if required.
4. Use slicers to interact with the dashboard.
5. Explore sales insights through visualizations.
